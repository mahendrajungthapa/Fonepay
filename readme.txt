About Fonepay Payment Gateway Plugin

Fonepay Payment Gateway is a WordPress plugin developed by Mahendra Jung Thapa. It allows WooCommerce users to accept payments via Fonepay, providing a secure and convenient payment option for customers.

For more information and updates, you can visit the GitHub repository at https://github.com/mahendrajungthapa/Fonepay or contact the author via Mahendra Jung Thapa's website at http://mahendrathapa.com.np/.
